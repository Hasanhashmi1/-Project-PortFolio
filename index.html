Project:TCN

TCN is a comprehensive full-stack web application I developed as a freelance solution to streamline payment collections and customer management for small businesses. Built with React.js (Frontend), Node.js + Express.js (Backend), and PostgreSQL (Database), the system provides field agents with tools to record cash/UPI payments while giving administrators complete oversight of operations. To maximize cost-efficiency for the business owner, I implemented a QR-based UPI payment system using React QR code generators, eliminating expensive payment gateway fees. The platform features: Real-time tracking of customer dues and agent collections Automated QR code generation with preset payment amounts Admin dashboard for monitoring active/inactive customers and approving new connections SweetAlert2 for clean, interactive notifications Responsive design ensuring accessibility on all devices By combining robust technologies with thoughtful UX design, TCN delivers an affordable, scalable solution that reduces operational costs while improving collection efficiency. The system's modular architecture allows for future expansion as business needs grow. (Private project - additional details available upon request.)
